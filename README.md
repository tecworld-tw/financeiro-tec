# TecWorld — Controle Financeiro

Sistema de controle financeiro e gestão de vendas da TecWorld.

## Tecnologias

- React
- Vite
- Tailwind CSS
- Shadcn UI
- TanStack Query
- Lucide React
- Sonner
- Vaul
- Zod

## Como rodar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Rode o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
